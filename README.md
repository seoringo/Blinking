 # Blinking
 １年後期制作のAndroidアプリ<br>
 ３人でのグループ開発

## 概要
ドライアイ防止アプリ<br>
顔検知(ML Kit)を用いて、ユーザーが長時間目を開いていたらフラッシュをまばたきをさせる<br>
フォードバックもある
